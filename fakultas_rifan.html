Fakultas di Universitas Perjuangan Tasikmalaya

Universitas Perjuangan Tasikmalaya (UNPER) memiliki beberapa fakultas unggulan yang menaungi berbagai program studi berkualitas dan relevan dengan kebutuhan industri modern.

Fakultas Keguruan dan Ilmu Pendidikan (FKIP)
Program Studi: S1 Pendidikan Guru Sekolah Dasar (PGSD), S1 Pendidikan Bahasa Inggris (PBI) Fakultas Ekonomi
Program Studi: S1 Manajemen, S1 Akuntansi Fakultas Teknik
Program Studi: S1 Teknik Informatika, S1 Teknik Sipil Fakultas Ilmu Kesehatan
Program Studi: S1 Farmasi Fakultas Pertanian
Program Studi: S1 Agroteknologi, S1 Agribisnis, S1 Peternakan

Setiap fakultas di UNPER berkomitmen untuk menghasilkan lulusan yang tidak hanya kompeten secara akademik, tetapi juga memiliki karakter kuat dan semangat perjuangan sesuai dengan nilai-nilai universitas.
